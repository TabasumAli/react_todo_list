# Todo List App

A simple todo list management application built with React.js.

## Features

- Add new todos
- Mark todos as complete/incomplete
- Delete todos
- Sorted todo list
- Responsive design

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn
